# 📄 Annex D — Output Interpretation Framework

---

## 🧭 Purpose

This annex defines how simulation results from the Sextant Orbital Resilience Framework are interpreted after cascade propagation (Annex C).

It converts raw system state into:

- resilience metrics  
- cascade severity classification  
- system health evaluation  

---

## 🧠 Core Concept

The system does not only simulate failure propagation.

It also:

> interprets system-wide behaviour into structured resilience intelligence.

This ensures simulation output is **actionable, comparable, and deterministic**.
